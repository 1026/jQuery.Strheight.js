jquery.Strheight.js

About

Straight + Strech + Height = Strheight

ラッパー等に依存せず1行づつ高さを揃える事が出来る。
1:2、1:3、2:3等複数要素で高さを揃えることも出来る。
併用も出来る。
僕の考えた最強の高さを揃えるJS

Usage

HTML

<script type="text/javascript" src="/vendor/jquery.min.js"></script>
<script src="/js/jquery.strheight.js"></script>

JavaScript（optionは初期値を記載）

$(function(){
  Strheight(
    ['.fuga','.hoge','.foo','.bar'], //1行づつ高さを揃えたい要素
    {'.piyo':'p'}, //1:2、1:3、2:3等複数要素で高さを揃えたい要素
    {
      checkInterval: 500, //フォントサイズの変更を監視するインターバル（不要な場合は0）
      liquid: false, //リキッド・レスポンシブ等横幅可変に対応するか
      divideEqually: true //複数要素で高さを揃えるとき追加する高さを当分割するか、最後の要素にまとめるか
    }
  );
});

1行づつ高さを揃えたい要素毎にcssセレクタを配列で指定する。
['.fuga, .hoge','.foo, .bar']のような指定も勿論可能。

1:2、1:3、2:3等複数要素で高さを揃えたい要素のcssセレクタを
{'ラッパー':'要素'}の形の連想配列で指定する。
詳しくはデモ。

1行づつと複数要素の併用も可能（物理的に不可能な指定でなかれば）。
どちらか片方だけ利用の場合は、配列を空に（[]or{}）。

リフレッシュ機能

要素内部をAjaxで読み込んだ時等のリフレッシュ機能は特段設けてはいないが、以下のような方法で対応可能。

var strheight = Strheight(
  ['.foo','.bar'],
  {'.piyo':'p'}
);

strheight.refresh();
